# 💰 デイユースホテル検索サイト - マネタイズ戦略

## 🎯 プロジェクト概要

- **タイプ:** ニッチ検索・マッチングサイト
- **ターゲット:** テレワーカー、カップル、観光客、出張ビジネスマン
- **市場規模:** デイユース市場は拡大中（テレワーク需要）
- **競合:** 楽天トラベル、じゃらん（機能の一部）、個別ホテルサイト

### Phase 3: 直接掲載契約（6-12ヶ月）

**ホテル掲載プラン:**

| プラン | 価格 | 内容 |
|--------|------|------|
| **Basic** | ¥5,000/月 | 通常掲載 |
| **Premium** | ¥15,000/月 | 上位表示 + バッジ |
| **Featured** | ¥30,000/月 | トップ表示 + 特集記事 |

**想定:**
- Basic: 30施設 × ¥5,000 = ¥15万円
- Premium: 15施設 × ¥15,000 = ¥22.5万円
- Featured: 5施設 × ¥30,000 = ¥15万円
- **月収: ¥52.5万円（掲載料）+ ¥20万円（広告・アフィリ）= ¥72.5万円**

**営業ターゲット:**
1. ビジネスホテルチェーン（東横イン、ルートイン等）
2. シティホテル（都心部）
3. カプセルホテル
4. ラブホテル（デイユース特化）

**予約手数料モデル:**
- 予約手数料: 10%
- 月間予約数: 500件
- 平均単価: ¥5,000
- 月間取扱高: ¥250万円
- **月収: ¥25万円（手数料）+ ¥72.5万円（掲載・広告）= ¥97.5万円**


## 🚨 リスクと対策

| リスク | 対策 |
|--------|------|
| SEO競合強い | ニッチキーワード特化 + コンテンツ量で勝負 |
| ホテル営業失敗 | SEO・広告収益先行、実績作ってから営業 |
| 予約システム開発コスト | Phase 3まで外部アフィリで実績作る |
| 法的リスク（旅行業法） | 予約仲介ではなく「情報提供+アフィリ」で開始 |

---

## 🎯 成功の鍵

1. **SEOに全力:** 最初6ヶ月はコンテンツ拡充
2. **ニッチ特化:** 大手に勝てないので「デイユース専門」で差別化
3. **ユーザー体験:** 検索→予約の導線を極限まで短く
4. **データ蓄積:** ユーザー行動分析→改善サイクル
